## Vakselista
Vaskelita er ment for kollektiver som sliter med å organisere vaskerutinene sine. Tanken er at du skal kunne dele
kollektivet inn i seksjoner og seksjonene inn i oppgaver. Hver uke skal applikasjonen tildele seksjonene som hver bruker skal vaske,
og ved slutten av uken skal brukeren få en mail om ikke alle oppgavene er gjort. Neste uke får alle brukerene en ny seksjon.

Vaskelista er skrevet i react/redux frontend og java spring boot backend.
